# MCRAMILEUX's Note Taker
11th Challenge: Express.js - Note Taker

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## Description
- This is a basic note taking application with a frontend starter code called the Note Taker which can be used to write and save notes. The application uses an Express.js backend and will save and retrieve note data from a JSON file.

## Installation
* Navigate to this [repository](https://github.com/mcramileux/mcramileux-note-taker).
* Open your terminal on your local machine and clone the repository.
* To install the npm packages, run npm i on your terminal.
* Once done, deploy the repository in [Heroku](https://rocky-fjord-38162.herokuapp.com).

## Usage
* Run VS Code's terminal by typing "npm i && npm start" commands to install the dependencies.
* Run "npm i uuid" to activate the uuid package that will provide the user the generated unique identifiers.
* Once successful on the following commands above, the user will know everything is properly done, this will appear in th terminal: "App listening at http://localhost:3001 ðŸš€".
* Copy the link http://localhost:3001 to the web browser.
* Since this challenge includes Heroku deployment, navigate to this [site](https://rocky-fjord-38162.herokuapp.com) to create and delete the notes.
